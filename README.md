# IronPulse: Gym Social Media Webapp

IronPulse is a social media web app for lifters to socialize, post PRs, and get workout advice from others. 

## Instructions for Building and Using

Steps to build and/or run the software:

1. Download and install VSCode
2. Download the files from GitHub
3. Run the server myfirst.js
4. Then follow the instructions to bring up the web-app (You will need your IP Address)

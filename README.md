# IronPulse: Gym Social Media Webapp

IronPulse is a social media web app for lifters to socialize, post PRs, earn badges, and get workout advice from others in the workout community. 

## Instructions for Building and Using

Steps to build and/or run the software:

1. Download and install VSCode
2. Install Node.js
3. Download the files from GitHub
4. Open a terminal in VS Code
5. Install express
6. Run the server myfirst.js
7. Then follow the instructions to bring up the web-app (You will need your IP Address)
   
Instructions for using the software:

1. On the Home page, create posts to share lifts or PRs
2. You can like posts
3. In the Profile page:
4. Log daily workouts:
   1. Select muscles groups
   2. Enter workout
   3. Save
5. You can see the history of workouts
6. Input PRs
